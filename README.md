
# Guardia 手机防火墙卫士

这是一个基于 React 和 Capacitor 构建的高保真手机防火墙演示应用。

## 功能特性
- **实时监控**：图形化展示系统保护状态。
- **规则管理**：支持黑名单、白名单以及短信关键字过滤。
- **安全日志**：记录所有被拦截的通话和短信详情。
- **模拟器**：内置模拟攻击测试工具。

## 如何获取 APK
1. 点击仓库顶部的 **Actions** 选项卡。
2. 选择 **Build Android APK** 工作流。
3. 在最新的成功运行记录中，找到底部的 **Artifacts** 区域下载。

## 技术栈
- 框架：React 19
- 样式：Tailwind CSS (Material 3 Design)
- 移动端引擎：Capacitor JS
- 自动化构建：GitHub Actions
